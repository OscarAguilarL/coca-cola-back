# CocaCola Project API

1. Clonar el proyecto

2. `yarn install`

3. Configurar variables de entorno

```
cp .env.example .env
```

4. Levantar la DB
```bash
docker-compose up
```

5. Levantar el proyecto

```
yarn start:dev
```
